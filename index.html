What's this for?

This website is intended to summarize all the Chemistry definitions and concepts in the Gr 11 Curriculum in a simple way. The list is by no means complete, and if you want to contribute, please refer to the contributing section. This is my first ever web project, and this site is a work in progress. If there are any problems, please give me feedback on how the site could be improved using the Feedback Form, or contact me using one of the methods below.

Contributing

Whether you want to add a definition, implement a new feature, or improve upon something that exists, simply fork this repo on GitHub, create a new branch, and submit a pull request. Don't know git or never heard of GitHub? You can still contribute! Simply submit any suggestions or feedback using the Feedback Form.

Anything else like this?

This is currently the only site I have made, more sites like Functions 11, Physics 11 will be created in the future, given that I have enough time.

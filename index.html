What's this for?

This website is intended to summarize all the Chemistry definitions and concepts in the Gr 11 Curriculum in a simple way. The list is by no means complete, and if you want to contribute, please refer to the contributing section.

Whether you want to add a definition, implement a new feature, or improve upon something that exists, simply fork this repo on GitHub, create a new branch, and submit a pull request.
